## 📌 Project Overview

This project delivers a comprehensive, end-to-end data transformation and business intelligence solution focused on the global data job market. The goal of this project was to establish a fully scaled analytics pipeline—handling raw, isolated data fragments, connecting them with a structured relational architecture, and deploying an interactive reporting interface.

The final production model enables stakeholders to interactively query and analyze high-level trends, including global job distributions, technology skill demands, and granular compensation patterns (both hourly and yearly) across distinct data science and analytics engineering roles.

## 🛠️ Project Implementation

### 1. Data Engineering (Power Query & M Language)
* **Dynamic Data Import:** Built an automated ETL pipeline that handles both folder-based batch imports and individual CSV file integrations, ensuring a unified data source.
* **M Language Scripting:** Utilized **M (Power Query Formula Language)** for advanced transformations, query optimization, and handling complex cleaning logic beyond standard UI capabilities.
* **Data Cleaning & Formatting:** Groomed data by removing redundancies, resolving inconsistencies, and standardizing data types across all imported datasets for consistent integration.

### 2. Data Modeling & Architecture
* **Star Schema Integration:** Leveraged a pre-structured Star Schema framework and extended the model with custom dimension tables to ensure efficient, high-performance data relationships.
* **Relationship Mapping:** Configured relationships to ensure seamless cross-filtering and accurate data propagation across the reporting model.
* **Model Responsiveness:** Optimized the structural hierarchy and data types within the model to guarantee high performance and fast interactivity during report usage.

### 3. DAX & Analytical Logic
* **Advanced Measure Authoring:** Developed an extensive library of **explicit DAX measures** for core KPIs, ensuring full calculation control and model performance optimization compared to implicit defaults.
* **Context & Relationship Intelligence:** Utilized DAX evaluation contexts to handle complex data relationships and cross-filtering, ensuring accurate metric calculation across the Star Schema.
* **Dynamic Interactivity:** Implemented Field Parameters through DAX to allow for dynamic axis and measure switching, providing users with high-end analytical flexibility and a tailored reporting experience.

### 4. Dashboard & Visualizations
* **Interactive Design:** Crafted a user-centric dashboard interface that ensures seamless navigation and data exploration.
* **Visual Formatting:** Applied professional UI standards, including consistent color themes, clear labeling, and intuitive layouts to enhance data storytelling.
